# Algorithms Practice for QA Automation & Technical Interviews

This repository is a structured collection of **algorithm exercises written in JavaScript**, designed to strengthen logical reasoning, data structures understanding, and problem-solving skills commonly evaluated in **QA Automation**, **SDET**, and **Software Engineering** interviews.

The repository is organized into **10 core algorithm categories**, each containing **20 exercises** with increasing difficulty.  
The goal is to train loop control, conditionals, string/array manipulation, hash maps, mathematical logic, and classic interview patterns.
